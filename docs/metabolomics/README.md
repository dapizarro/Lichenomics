# Metabolomics Workflow

## Overview

This workflow describes a reproducible strategy for processing lichen LC-MS/MS datasets.

### Objectives

* Feature detection
* Feature alignment
* Feature filtering
* Molecular networking
* Metabolite annotation
* Statistical analyses
* Ecological integration

---

## Input

Raw LC-MS/MS files

```text
mzML
mzXML
```

---

## Workflow

Raw Data
↓
MZmine
↓
Feature Table
↓
GNPS
↓
Molecular Networking
↓
Annotation
↓
Statistics
↓
Ecological Interpretation

---

## Outputs

* Feature table
* Annotated metabolites
* Molecular networks
* PCA
* PCoA
* PERMANOVA
* Differential metabolites

---

## Example Applications

* Chemotype comparisons
* Environmental gradients
* Species comparisons
* Symbiosis studies
