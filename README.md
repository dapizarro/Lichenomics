# 🧬 Lichenomics

### Reproducible Multi-Omics Framework for Lichen Symbiosis Research

Lichenomics is an open-source collection of reproducible workflows, notebooks and best-practice guidelines for studying lichen symbioses using modern multi-omics approaches.

This repository aims to facilitate the integration of microbiome profiling, metagenomics, comparative genomics, metabolomics and ecological data in lichen research.
