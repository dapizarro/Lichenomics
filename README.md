# 🧬 Lichenomics

## Reproducible Multi-Omics Framework for Lichen Symbiosis Research

Lichenomics is an open-source collection of reproducible workflows, tutorials, example datasets and best-practice guidelines for studying lichen symbioses using modern multi-omics approaches.

The framework aims to integrate:

* Metabolomics (LC-MS/MS)
* Microbiome profiling (16S rRNA)
* Metagenomics
* Comparative genomics
* Biosynthetic gene clusters (BGCs)
* Ecological metadata
* Multi-omics integration

to understand the ecological and evolutionary mechanisms underlying lichen symbioses.

---

## Current Modules

### Metabolomics

LC-MS/MS processing, molecular networking and ecological interpretation.

Status: ✅ Available

### Microbiome

Amplicon-based microbial community analyses.

Status: 🚧 Under development

### Metagenomics

Assembly, binning and functional annotation workflows.

Status: 🚧 Under development

### Comparative Genomics

Phylogenomics, orthology inference and evolutionary analyses.

Status: 🚧 Under development

### Multi-Omics Integration

Integration of ecological, metabolomic and genomic datasets.

Status: 🚧 Under development
